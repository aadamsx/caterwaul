Javascript without the noise

Introduction

Caterwaul is a Javascript-to-Javascript compiler implemented in Javascript. If this sounds pointless (it probably does), think of it as a way to convert Javascript that you wouldn't write into Javascript that you would. For example, nobody would write this:

let[x = 5] in x + 1

Using Caterwaul, however, you can define transformations for certain forms. Caterwaul's standard macro library transforms the above code into this:

(function (x) {return x + 1}).call(this, 5)

Caterwaul's transformation system is Turing-complete (it is very similar to Lisp macros, but slightly more expressive), and you can define macros that run arbitrary Javascript functions when they are invoked.

Getting Started

It's really easy to use Caterwaul. All you need to do for client-side code is drop this script tag into your code:

<script src='http://spencertipping.com/caterwaul/caterwaul.min.js'></script>

(You should probably mirror this script locally just in case my web host goes down.) After that you can wrap any top-level function (i.e. a function that isn't a closure) in a caterwaul call, like this:

caterwaul.clone('std')(function () {
  let[x = 5] in alert('x is #{x}');
})();

Caterwaul supports a bunch of cool macros and code transformations. Check out the Caterwaul offline compiler (which runs in your browser) for examples.
